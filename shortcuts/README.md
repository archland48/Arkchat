# Shortcuts 腳本說明

這些 Shortcuts 腳本可以幫助你在 iOS 和 Mac 上顯示每日經文。

## 使用方法

### iOS

1. **簡單版本** (`daily-verse-ios.shortcut`)
   - 獲取每日經文並顯示通知
   - 適合快速查看

2. **圖片版本** (`daily-verse-with-image.shortcut`)
   - 獲取每日經文並生成圖片
   - 保存到相簿
   - 可以用作鎖定畫面或桌面背景

### 導入步驟

1. 將 `.shortcut` 文件下載到你的 iPhone/Mac
2. 在「檔案」App 中打開
3. 點擊「分享」→ 「加入捷徑」
4. 編輯 Shortcut，將 `https://your-domain.com/api/daily-verse` 替換為你的實際 API URL

### 設置自動化

#### iOS

1. 打開「捷徑」App → 「自動化」
2. 點擊「+」→ 「創建個人自動化」
3. 選擇「特定時間」
4. 選擇每天的時間（建議 00:01）
5. 添加動作：運行剛才導入的 Shortcut
6. 關閉「執行前先詢問」

#### Mac

1. 打開「系統設定」→ 「隱私權與安全性」→ 「自動化」
2. 創建新的自動化
3. 設置觸發時間
4. 選擇運行的 Shortcut

## 自定義

### 修改 API URL

在 Shortcut 中找到「取得網頁內容」動作，修改 URL 為你的實際 API 端點。

### 修改圖片樣式

在「製作圖片」動作中，可以修改：
- 背景顏色
- 文字顏色
- 字體大小
- 圖片尺寸

### 修改通知內容

在「顯示通知」動作中，可以修改標題和內容格式。

## 注意事項

1. 確保你的 API 已經部署並可訪問
2. 確保設備有網路連接
3. 給予 Shortcuts 必要的權限（通知、照片等）
